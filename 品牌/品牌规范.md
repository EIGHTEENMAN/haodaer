## 品牌定位

童慧行包含两大站点，分别服务"学"与"行"两个场景：

| 站点 | 域名 | 定位 | 目标用户 |
|------|------|------|---------|
| 童慧行主站 | grandand.com | 儿童学习资源 + 益智挑战 | 儿童（学习）、家长（辅助） |
| 童慧行走天下 | travel.grandand.com | 真实亲子旅行攻略 | 家长（攻略）、儿童（互动） |

核心价值观：儿童体验第一、隐私保护、非商业化优先。

## 常用操作规范

1. **修改远程服务器文件**：编写 Python 脚本上传至 `/tmp/`，通过 `expect` 处理 SSH 密码 (`Hdet@2026`) 执行，构建后 `pm2 restart` 对应服务
2. **CSS/Tailwind 颜色修改**：主站用 CSS 类名（非 Tailwind），走天下用 Tailwind utility classes
3. **搜索 + 详情页逻辑**：搜索内容来自 auth-service 数据 (`data/search/`)，结果直接跳转子应用源 URL
